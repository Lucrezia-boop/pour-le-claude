Executive Installation Guide

ComfyUI with Hunyuan3D 2.1
Purpose: This guide will install a complete AI-powered 3D generation system (ComfyUI + Hunyuan3D 2.1) on a fresh Windows machine with minimal user interaction.

What This Software Does

ComfyUI with Hunyuan3D 2.1 is an AI system that:

  • Converts images into 3D models
  • Generates professional PBR textures (used in architecture, games, film)
  • Exports models in industry-standard formats (GLB, OBJ)
  • Runs locally on your computer (no cloud/subscription needed)

System Requirements

Hardware:

  • Windows 10 or Windows 11 (64-bit)
  • NVIDIA GPU with at least 12GB VRAM (RTX 3090/4090 recommended)
  • 16GB+ system RAM (32GB recommended)
  • 100GB free disk space (50GB for models, 50GB for software)

Software Prerequisites (to be installed):

None initially - this guide covers everything from scratch

This installation has TWO PHASES:

PHASE 1: Foundation Software (Manual Installation) ~1 hour

These are industry-standard tools that require one-time setup:

  1. Miniconda - Python environment manager (10 minutes)
  2. CUDA Toolkit 12.8 - NVIDIA GPU computing platform (15 minutes + restart)
  3. Visual Studio 2022 Build Tools - Microsoft C++ compiler (20 minutes)
  4. Git for Windows - Version control system (5 minutes)
  5. Automated Verification - Run verify_prerequisites.bat (handles all checks and config)

Everything is fully automated - just double-click and wait:

  • Creates isolated Python environment
  • Downloads and installs ComfyUI
  • Installs Hunyuan3D extension with all dependencies
  • Compiles custom GPU acceleration code
  • Verifies installation
  • Creates desktop shortcut

Troubleshooting

Installation Issues

üí° First Step for Any Problem:

Re-run verify_prerequisites.bat from bundle_setup-files/2_Registry_And_Verification/

This script automatically checks and fixes most common issues. It will tell you exactly what's wrong and what needs to be fixed.

Problem: verify_prerequisites.bat shows [FAIL] for Python or Conda

Solution:
  • Most likely cause: Miniconda not added to PATH during installation
  • Fix: Uninstall Miniconda from Windows Settings ‚Üí Apps
  • Reinstall and make sure to CHECK "Add to PATH" checkbox
  • Restart computer
  • Run verify_prerequisites.bat again

Problem: verify_prerequisites.bat shows [FAIL] for CUDA

Solution:
  • Most likely cause: CUDA not installed or wrong version
  • Fix: Reinstall CUDA 12.8 from 1_Installers/
  • Use "Custom (Advanced)" installation
  • Check ONLY "CUDA Toolkit 12.8"
  • Restart computer
  • Run verify_prerequisites.bat again

Problem: verify_prerequisites.bat shows [FAIL] for Visual Studio

Solution:
  • Most likely cause: Missing individual components
  • Fix: Re-run vs_BuildTools.exe from 1_Installers/
  • Choose "Modify" if already installed
  • Make sure ALL 8 individual components are checked (see Step 3)
  • Run verify_prerequisites.bat again

Problem: verify_prerequisites.bat shows [FAIL] for disk space

Solution:
  • Need: 50GB+ free space on C: drive
  • Fix: Free up disk space
  • Use Disk Cleanup: Search "cleanmgr" in Start menu
  • Consider moving personal files to external drive
  • Run verify_prerequisites.bat again

Problem: Automated installer fails during compilation (Steps 10-11)

Solution:
  • Most likely cause: Visual Studio Build Tools not fully configured
  • Fix: Re-run verify_prerequisites.bat first
  • If that passes, reinstall VS Build Tools with all 8 individual components
  • Re-run the automated installer (it will resume where it left off)

Runtime Issues

Problem: Console shows errors on startup

Solution:
  • Check: ComfyUI-Hunyuan3D-2_1 nodes loaded correctly
  • Look for: [OK] ComfyUI-Hunyuan3D-2_1 loaded in console
  • If missing, re-run installer: python install_comfyui_hunyuan3d.py --reinstall-nodes

Problem: "CUDA out of memory" error

Solution:
  • Your GPU has insufficient VRAM
  • Try: Close other GPU-intensive applications
  • Minimum required: 12GB VRAM

Problem: Models not downloading

Solution:
  • Check: Internet connection and firewall settings
  • Models download to: C:\Users\[YourName]\.cache\hy3dgen\tencent\Hunyuan3D-2.1\
  • Can manually download and place in this folder

Problem: Slow performance (>15 minutes per model)

Solution:
  • Check: CUDA is being used (not CPU)
  • In console, look for: "CUDA available: True"
  • If False, reinstall CUDA Toolkit 12.8
